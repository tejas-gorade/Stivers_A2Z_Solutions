# Step 3.1: Arrays - Easy Problems

This folder contains optimized C++ solutions for fundamental array problems. These problems form the core of technical interviews and focus on achieving linear time complexity $O(n)$ while maintaining minimal space complexity $O(1)$.

Also the Brute and Better Approach.(if needed)

### 💡 Key Learning Principles
* **XOR Property:** Efficiently finding missing or unique elements without overflow.
* **Two-Pointer Technique:** Crucial for merging sorted data and in-place array manipulation.
* **Optimal Space:** Moving from $O(n)$ hashing solutions to $O(1)$ sliding window techniques for non-negative arrays.
